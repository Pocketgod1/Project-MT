  - Local hardware store → bolts/screws/springs
  - Amazon → ABS filament (1-2 day delivery)

### Week 2: PARTS ARRIVE
- [ ] **Receive shipments**:
  - Verify servo models (HK15298B or JX PDI-6221MG)
  - Check filament color/type (natural ABS)
  - Organize parts by category
  - ⚠️ **Servo check**: Confirm correct servo specs before printing
- [ ] **Download CAD files**:
  - Visit: https://github.com/InMoov/InMoov
  - Download hand/finger 3D models (STL files)
  - If using JX PDI-6221MG, verify mounting hole compatibility

### Week 3-4: 3D PRINTING & ASSEMBLY
- [ ] **Calibrate 3D printer** for ABS (100°C bed)
- [ ] **Start printing finger parts** (~40-60 hours total)
- [ ] **Test servos individually** (use servo tester or Arduino)
- [ ] **Assemble hand mechanism**:
  - Mount servos in printed housing
  - Route fishing line tendons
  - Install return springs
  - Test finger movement

## ⚠️ CRITICAL REMINDERS

🔴 **DO NOT SKIP THESE**:
1. ✅ Order from **Hobbyking** for HK15298B (most reliable source)
2. ✅ Buy **HK15298B** (NOT HK15298 - old version)
3. ✅ Use **ABS filament** (NOT PLA - too weak for servos)
4. ✅ Get **0.8mm braided** fishing line (NOT monofilament)
5. ✅ Buy **extension springs** (finger return mechanism)
6. ✅ Test servos individually BEFORE assembly
